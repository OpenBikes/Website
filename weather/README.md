# Weather Data

## OpenWeatherMap

We make a call to OpenWeatherMap API.

- [Website](http://openweathermap.org/)
- JSON
- API key

## Parameters of API respond for current weather

    {
    	'datetime': timestamp (ISO format)
        'weather': string,
        'temp': float,
        'humidity': float,
        'pressure': float,
        'description': string,
        'wind_speed': float,
        'wind_speed': float,
        'rain_vol': float,
        'snow_vol': float,
        'clouds_pct': float
    }
